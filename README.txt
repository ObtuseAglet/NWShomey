NWS Weather Integration for Homey

This app integrates the National Weather Service (NWS) API into Homey, allowing you to create automations based on official weather data from NOAA.

Features (Planned):
- Real-time weather alerts and warnings
- Current conditions and forecasts
- Severe weather notifications
- Custom location support
- Flow cards for weather-based automations

The National Weather Service provides free, reliable weather data for the United States and its territories. This app brings that data directly into your Homey for powerful weather-based home automation.

Note: This is an initial release with basic functionality. More features will be added in future updates.

Data Source: National Weather Service (NOAA) - https://api.weather.gov/
